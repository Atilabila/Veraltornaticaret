## Definition of Done
Bir iş **ancak şu şartlarla DONE sayılır:**

- 320 / 375 / 768 / 1024 / 1440 ekranlarda test edildi
- Zoom %50 ve %30’da layout çökmedi
- CTA kaybolmuyor
- Ürün kartları standart
- “Dev boşluk” yok

---

## Forbidden Patterns
- `position:absolute` ile ana layout
- Rastgele `margin-top: 87px` gibi değerler
- PX-only responsive tasarım
- “Sonra bakarız” yaklaşımı
